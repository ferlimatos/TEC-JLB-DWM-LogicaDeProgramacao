# Sobre
Esta pasta foi criada com o propósito de servir como um caderno de prática complementar para reforçar os conceitos de Algoritmos e Lógica de Programação, utilizando Python.

## Organização da Pasta
Os exercícios estão separados em subpastas, agrupados pelo conceito principal que praticam:

| Subpasta | Objetivo Principal|
| :--- | :--- |
| [listas](/lista-exercicios-especiais/listas-python/) | Prática intensa de todos os métodos de manipulação de listas. |
| [variaveis-e-tipos](/lista-exercicios-especiais/variaveis-e-tipos-python/) | Foco em I/O (input, print) e conversão de tipos de dados. |
| [operadores-aritmeticos](/lista-exercicios-especiais/operadores-aritmeticos-python/) | Foco em fórmulas, precedência e cálculos matemáticos. |
| [condicionais-decisao](/lista-exercicios-especiais/condicionais-decisao-python/) | Foco na lógica de if/elif/else e condicionais aninhadas. |
| [estruturas-de-repeticao](/lista-exercicios-especiais/estruturas-de-repeticao-python/) | Foco na implementação de for, while, Contadores e Acumuladores. |
